# War of Dots Lite (Python)

Um mini jogo inspirado em **War of Dots**, projetado para rodar bem em hardware simples (como Intel Graphics 2000). O objetivo é dominar a maior parte dos pontos antes do tempo acabar.

## Controles

- **WASD**: mover o seu ponto.
- **Espaço**: soltar um pulso (converte pontos próximos).
- **R**: reiniciar quando o jogo termina.

## Como jogar

1. Instale as dependências.
2. Execute o jogo.

```bash
pip install -r requirements.txt
python main.py
```

## Regras rápidas

- Capture pontos ficando perto deles ou usando o pulso.
- Se você controlar 70% dos pontos, vence.
- Caso o tempo acabe, vence quem tiver mais pontos.

Divirta-se!
